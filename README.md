# Seasonal Outfit Recommender

이 프로젝트는 사용자의 스타일 선호도를 입력받아 선택된 계절에 맞는 의상을 추천합니다. **SentenceTransformer**와 **Hugging Face Transformers**를 활용하여 텍스트 유사도를 분석하고, 사용자의 입력과 가장 잘 어울리는 의상을 제안합니다.
